#### Go commands

```go
// compiles and executes one or more files
go run <file-name>.go

// compiles the code
go build <filename>.go

// formats the code
go fmt 

// compiles and installs a package
go install <package-name>

// downloads the source-code
go get 

// to run tests
go test
```
